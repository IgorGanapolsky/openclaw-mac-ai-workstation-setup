# SOP: Multi-Touch Prospecting (Blount/Voss/Cialdini)
*Framework: Fanatical Prospecting & Never Split the Difference*

## 1. The 30-Day Prospecting Plan
Never let the pipeline go cold. Every 30 days must have:
- **Daily Outreach:** Use the "Prospecting Bank" for LinkedIn/X/Email.
- **Law of Familiarity:** Engagement (Likes/Comments) before the DM.
- **Multi-Touch:** 
  1. Day 1: X/LinkedIn Mention/Comment.
  2. Day 2: DM/Email using Voss's Calibrated Questions.
  3. Day 5: Follow-up with a "Value Asset" (e.g., a short loom or diagnostic).

## 2. Tactical Empathy & Labels (Voss)
- **Labeling:** "It seems like you're losing leads to competitors because your response time is too slow."
- **Calibrated Questions:** "How is your team supposed to handle missed calls while they are on-site?"
- **The "No" Strategy:** "Is it a ridiculous idea to see if we can catch $5k in missed revenue for you this week?"

## 3. Psychological Persistence (Cialdini)
- **Reciprocity:** Give a free diagnostic first.
- **Social Proof:** "I just set this up for a roofer in Miami and they caught 3 leads in 24 hours."
- **Scarcity:** "I only have bandwidth for 2 more 'Forward Deployments' this week."

## 4. Implementation Task
- **Action:** Generate 5 personalized messages daily using these frames.
- **Log:** Record every outcome in `lead-log.md`.
